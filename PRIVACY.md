# Privacy

Traffic Lights+ does not collect, store, or transmit personal data. It has no analytics, advertising, account system, or network code.

The app requests macOS Accessibility permission only to:

- locate the currently focused window;
- read that window's position and size;
- invoke its standard close, minimize, and zoom buttons.

All processing happens locally on the Mac. Preferences are stored in the user's standard macOS preferences database.
